# Home Assistant Add-ons by Delido

A collection of Home Assistant add-ons with sidebar (Ingress) integration.

## Installation

1. Go to **Settings → Add-ons → Add-on Store**
2. Click the three-dot menu (⋮) → **Repositories**
3. Add: `https://github.com/Delido/hassio-addons`
4. Install any add-on from the list below

## Available Add-ons

### Wallos

Self-hosted subscription and payment tracker.

- Ingress (sidebar integration)
- Persistent data in `/share/wallos`
- [Documentation](wallos/README.md)

### MinIO

S3-compatible object storage server.

- Ingress (sidebar integration)
- S3 API on port 9000
- Persistent data in `/share/minio`
- [Documentation](minio/README.md)
